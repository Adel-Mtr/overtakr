# Overtakr Frontend

Next.js app for Overtakr's race strategy intelligence experience.

## Run locally

```bash
npm install
cp .env.example .env.local
npm run dev
```

## Environment

```env
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

## Main UX areas

- Strategy Builder
- Race Intelligence (pit windows, weather, safety car context)
- Strategy Delta Charts
- Driver Digest
- Overtake Intelligence
